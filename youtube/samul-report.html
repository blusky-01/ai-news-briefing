성공공식 문서화

이 채널의 본질은 생활밀착형 정보를 사물이 인간을 혼내는 캐릭터 쇼츠 포맷으로 바꿔, 유용함과 웃김을 동시에 주는 구조입니다.
1. 주제 선택
누구나 매일 겪는 사소한 실수. 음식 보관, 주방 세척, 건강/위생, 자동차/안전처럼 즉시 공감 가능한 소재.
2. 제목 구조
매일, 헷갈리는, 실수, 치명적, 망치는, 무서운 같은 경고형 표현 반복.
3. 화자 설정
사물/대상이 인간에게 직접 말하고 타박하는 캐릭터 구조. 정보 채널이면서 동시에 캐릭터 체험 채널.
4. 스크립트 구조
대상 제시 → 문제점 설명 → 해결책 제시 → 팁 추가
5. 성과 주제
건강/위생 + 음식/보관 + 주방/조리 축이 강함.
6. 확산 동력
유용함보다 혼나서 웃긴 경험이 더 강한 확산 포인트.
7. 운영 방식
카테고리는 넓지만 메시지는 하나: 너 지금 잘못 알고 있고, 매일 하는 습관이 문제이며, 바로 고칠 수 있다.

신규 채널 포지셔닝 3안 설계

A안. 냉정한 사물 연구소

장점: AI/과학 느낌 강화

리스크: 너무 딱딱하면 대중성 약화

B안. 잔소리하지만 유능한 생활코치

장점: 대중성 높음

리스크: 원조 채널과 유사해질 수 있음

C안. 도발적인 현실 팩폭 채널

장점: 확산력 강함

리스크: 선 넘으면 반감 커짐

대사 패턴 분석

사물이 자기소개를 하고, 인간의 잘못을 비꼬며 지적한 뒤, 바로 고칠 행동을 명령하는 쇼츠형 잔소리 문법이 핵심입니다.
기본 구조
자기소개 → 잘못 지적 → 위험/손해 강조 → 해결책 제시 → 비꼼/CTA 마무리
자주 쓰는 도입
나 ~인데, 왜 그렇게 하냐, 그걸 아직도 모르냐, 제발 ~하지 마
문장 리듬
질문형 → 면박형 → 명령형. 짧은 문장 블록을 빠르게 연타.
말투 본질
화남 60 / 비꼼 25 / 정보 15. 혼내지만 결국 돕는 톤.
자주 쓰는 비유
세균 파티, 비싼 맹물, 재앙, 사육장, 무덤 간다 같은 과장 비유.
생성 공식
나 [대상]인데 + 왜 [실수 행동] 하냐 + 그러면 [손해] 생긴다 + 제발 [수정 행동] 해라 + [비꼼/CTA]

샘플 대본 생성 테스트

샘플 1. 냉장고 문칸에 두면 안 되는 것들
나 우유인데 왜 자꾸 문칸에 꽂아 두냐? 문 열릴 때마다 온도 출렁여서 제일 먼저 상하는 건 나라고. 오래 먹고 싶으면 안쪽 깊숙한 칸에 넣어. 배탈 나서 화장실이랑 친구 먹기 싫으면 똑바로 보관해.
샘플 2. 전자레인지로 음식 데울 때 하는 실수
나 전자레인지인데 음식 가운데만 산처럼 쌓아 놓고 왜 차갑다고 화내냐? 열이 고르게 안 돌면 겉만 뜨겁고 속은 냉장고지. 넓게 펴고 중간에 한 번 섞어. 맨날 데웠다면서 차갑다고 놀랄 거면 나도 억울하다.
샘플 3. 욕실에서 칫솔 망치는 습관
나 칫솔인데 젖은 채로 컵 바닥에 처박아 두지 마. 물기 못 마르면 세균이랑 곰팡이한테 월세 내주는 거야. 쓰고 나면 털어 말리고 통풍되는 데 세워 둬. 입안에 세균 바르고 개운한 척하지 말고.
샘플 4. 자취방에서 밥솥 보관하는 법
나 전기밥솥인데 밥 퍼놓고 뚜껑 닫아 놓은 채 방치하냐? 김이랑 수분 갇히면 냄새 배고 물때 끼기 딱 좋아. 내솥 비우고 물받이부터 닦아. 밥맛 망치고 싶지 않으면 귀찮아도 순서 좀 지켜.
샘플 5. 운동화 세탁 후 말리는 실수
나 운동화인데 젖은 채로 구석에 던져 놓지 마. 통풍 안 되면 냄새는 기본이고 속까지 축축해서 곰팡이 놀이터 된다. 신문지 넣고 그늘에서 말려. 발냄새 제조 공장 만들고 싶지 않으면 좀 신경 써.
